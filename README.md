AI2PNG
======

AI2PNG is simple little script for batch converting a folder of Adobe Illustrator *.ai files to transparent 24 bit PNG files.

To install simply copy this script into /Presets/Scripts in you Adobe Illustrator application folder, then start the Adobe Illustrator program. The script will be accessible in the File > Scripts menu.

If you wish to modify the script to control the export path, or file type the code is commented out, and can be easily found and uncommented in the file.

Compatibility
======

This should work with Adobe Illustrator CS5, CS6, and CC on both Windows & Mac.

Credits
=======

AI2PNG is a modified version of the Adobe AI to PDF script, with a couple of enhancements to make it dead easy to use.
